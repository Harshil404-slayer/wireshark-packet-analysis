## 📡 Wireshark Packet Analysis

## 📌 Overview
This project demonstrates network traffic analysis using Wireshark, a powerful packet analyzer used to capture and inspect data packets in real time.

## 🎯 Objectives
Capture live network traffic
Analyze common protocols (TCP, HTTP, DNS)
Understand packet structure and communication flow
Detect basic network anomalies

## 🛠️ Tools Used
Wireshark
Windows/Linux OS
Network Interface (Wi-Fi/Ethernet)

## ⚙️ Methodology
Start packet capture on an active network interface
Apply filters (e.g., http, dns, tcp)
Inspect packet details (IP, protocol, ports)
Analyze communication patterns and identify anomalies

## ⚠️ Ethical Considerations
Capture traffic only on authorized networks
Avoid accessing sensitive data
